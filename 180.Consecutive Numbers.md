# Question
[180. Consecutive Numbers](https://leetcode.com/problems/consecutive-numbers/description/)

## Answer
[Submission](https://leetcode.com/problems/consecutive-numbers/submissions/1867449689/)
```sql
with cte as (
    SELECT *, LEAD(num, 1) OVER(ORDER BY id) next_1, LEAD(num, 2) OVER(ORDER BY id) next_2
    FROM Logs    
)

SELECT distinct num as ConsecutiveNums
FROM cte
WHERE num = next_1
AND num = next_2
```
